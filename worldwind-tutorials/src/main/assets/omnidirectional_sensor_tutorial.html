OmnidirectionalSensor Tutorial

Demonstrates how to create a OmnidirectionalSensor and add it to the globe

This tutorial demonstrates the creation of an OmnidirectionSensor to the globe. In addition to the sensor, a placemark indicating the position of the sensor is added to provide visual context.

The regions colored green indicate they are visible from the position provided to the OmnidirectionSensor. Regions colored gray are occluded by terrain from the provided position. This tutorial uses a sensor range of 10,000 meters.

Example

OmnidirectionalSensorFragment.java

The OmnidirectionSensorFragment class extends the BasicGlobeFragment and overrides the createWorldWindow method. An OmnidirectionalSensor object with specified position and range is then created and added to the globe.
